# Life Knowledge Base

Durable, searchable memory for long-lived context.

## Directory structure
- `daily/` — daily consolidation notes (`YYYY-MM-DD.md`)
- `projects/` — project-specific status, decisions, and next actions
- `areas/` — ongoing responsibility areas and standards
- `resources/` — reusable references, playbooks, and how-tos
- `tacit/` — preferences, operating norms, security boundaries
- `templates/` — note templates for consistent entries

## Starter templates/files
- `templates/daily-template.md`
- `tacit/preferences.md`
- `tacit/security.md`
- `projects/_index.md`

## Rules
- Never store secrets (tokens/passwords/private keys).
- Do not copy full transcripts.
- Store only durable facts: decisions, procedures, preferences, and next actions.
